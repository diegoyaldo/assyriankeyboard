Assyrian Keyboard

Download the Windows Keyboard Layout

Install the Assyrian Syriac keyboard for Windows and switch between English and Assyrian instantly using your keyboard. Please note that at this time, this keyboard is only avaliable for Windows users. We apologize for any inconvenience!

Installation Instructions:

  1. Download the span class="highlight"> ZIP file below.
  2. Extract the folder.
  3. Run setup.exe.
  4. Complete the installer.
  5. Restart your laptop or PC.
  6. Press Windows + Space to switch keyboards.
  7. Make sure English (United States) is also installed.
  8. Check in: Settings → Time & Language → Language & Region
